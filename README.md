### オーバーライド
親クラスと同じ名前のメソッドを子クラスに定義すると、子クラスのメソッドが優先して使用される。
これは、子クラスのメソッドが親クラスのメソッドを上書きしていることから、オーバーライドと呼ばれる。
```
class Ahoge {
  info() {
  *******
  }
}

class Bhoge extends Ahoge {
  info() {
  *******
  }
}

const bhoge = new Bhoge("*******", "*******");
dog.info();

```
▲ 子クラスBhogeのinfoメソッドが呼び出される（オーバーライド）
